Linux常用目录处理命令

发表于 2018-07-21 | 分类于 Linux

常识

/root root用户的家目录

/home/user1 其他用户的默认家目录

# 超级用户提示符

$ 普通用户提示符

命令的基本格式

命令 「选项」 「参数」

常用的目录处理指令

pwd

查看当前工作目录的完整路径

ls

查看目录列表

-a 查看所有文件

-l 以长格式显示目录下内容列表，可简写为ll

文件类型：- 文件 d 目录 l软链接文件

u所有者 g所属组 o其他人

r读 w写 x执行

mkdir

创建目录

-p 递归创建

cd

进入目录

cd ~ 进入当前用户的家目录

cd - 进入上次目录

cd .. 进入上一级目录

cd . 进入当前目录

rmdir

删除空目录

rm

删除文件

rm -r 删除目录

rm -rf 强制删除文件/文件夹

cp

复制文件

cp -r 复制目录

cp -p 连带文件属性复制

cp -d 若源文件是链接文件，则复制链接属性

-a 相当于-pdr

mv

剪切或改名，如果源文件和目标文件在同一级目录下则是改名
