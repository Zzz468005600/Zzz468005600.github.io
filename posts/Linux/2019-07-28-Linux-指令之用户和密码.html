Linux 指令之用户和密码

发表于 2019-07-28 | 分类于 Linux

1，修改密码的指令为：passwd

按照这个命令可以输入新密码。

2，创建用户的指令为：useradd

执行这个指令，一个用户就会被创建了。接下来可以自己调用 passwd stone 来设置密码，然后再登陆。切换用户可以使用 su stone 命令。

Linux 里是 “命令行 + 文件” 的模式。对用用户管理来说，我们通过命令创建的用户，实际上是放在 /etc/passwd 文件里的。这是一个文本文件。我们可以通过 cat 指令，将内容输出再命令行上。

再 /etc/passwd 文件里，我们可以看到 root 用户和我们刚才创建的 stone 用户。x 的地方应该是密码，密码不能放在这里，不然大家都能知道了。接下来的是用户 ID 和组 ID 。

/root 和 /home/stone 分别代表 root 用户和 stone 用户的主目录。主目录是指用户登陆进系统之后默认的路径。

/bin/bash 文件是用于配置登陆后的默认交互命令行的，Linux 登陆后的交互命令行是一个解析脚本程序，这里的配置是 /bin/bash。

3，删除用户

userdel stone 删除用户 stone。

4，用户组

Linux 中用户是分组的，前面的例子创建用户并没有指定哪个组，默认就会创建一个同名的组。

如果要再创建用户的时候指定组，可以使用 useradd stone1 -g testgroup 来指定新用户所在的组。但这边 -g 后面跟的组别名称一定是已经创建好的，创建组的命令是 groupadd testgroup 。创建的组别信息存储再 /etc/group 这个文件里，可以使用 cat 指令输出到命令行中。

上面的第一个字段 root，stone，testgroup 分别是组别的名称，testgroup 就是我们刚才创建的组；第二个字段 x 代表用户组的密码；第三个字段代表用户组的 ID ，可以和上面 /etc/passwd 文件中的用户组 ID 对应起来。
